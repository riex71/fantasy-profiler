# 판타지 세계 속 나의 캐릭터

성향 기반 이세계 캐릭터 프로파일러를 구현한 Streamlit 웹 애플리케이션입니다.

이 프로젝트는 중간고사 대체 과제를 위해 제작되었으며, **로그인 기능**, **캐싱 기능**, **퀴즈 기능**을 포함합니다.  
사용자는 다양한 상황형 질문에 답하면서 자신이 어떤 세계관, 역할군, 태그를 가진 캐릭터인지 확인할 수 있습니다.

- 학번: 2022204055
- 학부: 정보융합학부
- 이름: 윤동현

---

## 1. 프로젝트 소개

이 앱은 사용자가 여러 개의 상황형 질문에 응답하면, 응답 결과를 바탕으로 아래 요소를 조합해 하나의 캐릭터 리포트를 생성합니다.

- 세계관 1개
- 역할군 1개
- 보조 태그 2개
- 프롤로그
- 진행 조언

이 프로젝트는 단순한 성격검사 형태가 아니라,  
**판타지 세계에 떨어졌을 때 나는 어떤 방식으로 움직이고 살아남을 사람인가**를 중심으로 설계한 성향 기반 캐릭터 리포트입니다.

사용자는 직접 문항에 응답해 결과를 확인할 수 있고,  
랜덤 선택 기능을 통해 샘플 결과를 빠르게 확인할 수도 있습니다.

## 2. 주요 기능

### 로그인 기능
- 아이디와 비밀번호를 입력하여 로그인할 수 있습니다.
- 로그인 성공/실패에 따라 다른 메시지를 제공합니다.
- 로그인 상태는 `st.session_state`를 통해 유지됩니다.

### 캐싱 기능
- 질문 데이터, 사용자 데이터, 세계관 데이터, 역할군 데이터, 결과 텍스트 데이터를 불러오는 함수에 `st.cache_data`를 적용했습니다.
- 같은 JSON 파일을 반복해서 읽지 않도록 하여 앱 실행 효율을 높였습니다.

### 퀴즈 기능
- 총 60문항의 상황형 질문으로 구성되어 있습니다.
- 사용자는 각 문항에서 4개의 선택지 중 하나를 고르게 됩니다.
- 각 선택지는 세계관, 역할군, 태그, 성향 보정 점수에 연결되어 있습니다.
- 모든 응답이 끝나면 점수를 누적하여 최종 결과를 계산합니다.

### 랜덤 선택 결과 보기
- 로그인 후 메뉴에서 랜덤 선택 기능을 사용할 수 있습니다.
- 이 기능은 60문항의 선택지를 자동으로 랜덤 선택한 뒤 실제 점수 계산을 수행합니다.
- 데모 시연이나 빠른 결과 확인용으로 활용할 수 있습니다.

### 결과 리포트 기능
- 선택된 세계관
- 대표 역할군
- 세계관별 호칭
- 보조 태그 2개
- 조합형 프롤로그
- 진행 조언
- 결과 텍스트 다운로드 기능

### 캐릭터 아카이브 기능
- 사용자가 세계관, 역할군, 태그를 직접 선택해 결과를 미리 볼 수 있습니다.
- 실제 점수 계산 없이 다양한 조합을 프리뷰하는 용도로 사용할 수 있습니다.

### 결과 시각화 기능
- 세계관 점수
- 역할군 점수
- 상위 태그 점수
- 빅파이브 보정 점수(참고용)

결과 화면에서는 Plotly 그래프를 활용해 주요 점수를 시각적으로 확인할 수 있습니다.

## 5. 데이터 구성

### questions.json

* 총 60개의 문항이 저장되어 있습니다.
* 각 문항은 문항 번호, 섹션 번호, 질문 내용, 4개 선택지, 점수 매핑 정보를 포함합니다.
* 각 선택지는 세계관 / 역할군 / 태그 / 빅파이브 보정 점수에 연결됩니다.

### worlds.json

* 6개의 세계관 정보를 저장합니다.
* 각 세계관의 이름, 한 줄 요약, 상세 설명이 포함됩니다.

### role_titles.json

* 기본 역할군 이름과 세계관별 역할군 호칭을 저장합니다.
* 태그 한글 이름도 함께 저장합니다.

### result_texts.json

* 프롤로그와 진행 조언 생성을 위한 텍스트 조각을 저장합니다.
* 세계관 문장, 역할군 문장, 태그 문장, 조언 문장, 특수 조합 override 문장을 포함합니다.

### users.json

* 로그인용 사용자 정보를 저장합니다.
* 아이디, 비밀번호, 이름을 포함합니다.

## 6. 결과 생성 방식

사용자의 응답을 바탕으로 각 선택지에 연결된 점수를 누적합니다.

앱은 아래 순서로 결과를 생성합니다.

1. 문항 응답 수집
2. 세계관 점수 계산
3. 역할군 점수 계산
4. 태그 점수 계산
5. 빅파이브 보정 점수 계산
6. 세계관 1개, 역할군 1개, 태그 2개 선택
7. 조합형 프롤로그 생성
8. 진행 조언 생성

즉, 결과는 단순히 하나의 유형을 고정적으로 출력하는 것이 아니라
여러 선택의 조합을 바탕으로 하나의 **판타지 캐릭터 리포트**를 생성하는 방식입니다.

---

## 7. 캐싱 설명

이 프로젝트에서는 Streamlit의 `st.cache_data`를 활용하여 아래 데이터를 캐싱합니다.

* 질문 데이터 (`questions.json`)
* 사용자 데이터 (`users.json`)
* 세계관 데이터 (`worlds.json`)
* 역할군/태그 데이터 (`role_titles.json`)
* 결과 텍스트 데이터 (`result_texts.json`)

이렇게 캐싱을 적용한 이유는,
앱이 다시 실행되거나 페이지가 새로고침되더라도 같은 JSON 파일을 반복해서 읽지 않도록 하기 위해서입니다.

특히 질문 데이터와 결과 텍스트 데이터는 앱 실행 중 여러 번 참조되므로,
캐싱을 통해 불필요한 파일 입출력을 줄이고 앱 실행을 더 안정적으로 유지할 수 있습니다.

## 8. 앱 사용 흐름

1. 첫 화면에서 학번과 이름을 확인합니다.
2. 로그인 정보를 입력하여 로그인합니다.
3. 메뉴에서 직접 검사 또는 랜덤 선택 결과 보기를 선택합니다.
4. 직접 검사 시 60문항에 응답합니다.
5. 결과 화면에서 세계관, 역할군, 태그, 프롤로그, 진행 조언을 확인합니다.
6. Plotly 그래프로 점수를 확인할 수 있습니다.
7. 필요하면 결과를 텍스트 파일로 다운로드할 수 있습니다.
8. 캐릭터 아카이브에서 원하는 조합의 결과를 직접 미리 볼 수 있습니다.

## 9. 데이터 검증 방법

JSON 데이터 구조가 올바른지 확인하기 위해 `check_data.py`를 함께 포함했습니다.

실행 방법:

```bash
uv run python check_data.py
```

정상이라면 다음 메시지가 출력됩니다.

```text
모든 JSON 파일 검사를 통과했습니다.
```
